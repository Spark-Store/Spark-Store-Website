## 团队介绍
由深度科技论坛（bbs.deepin.org）大佬 shenmo 发起的星火工作组致力于丰富 Linux 生态，
取“星星之火，可以燎原”之意，组建了星火应用商店。
我们是个年轻的团队，大部分成员是 00 后，有小学生，中学生，大学生，来自五湖四海。
尽管我们年龄不同，资历不同，但是我们努力建设 Linux 生态的目标是相同的，
我们有信心、也有勇气做到这些！
